SER120 Lab Syllabus

Course Material

Primary Resources

  • Reading Assignments: I may use several sources for reading assignments:
    • Slides: I have slides created by professors who have taught this class in the past, which I will post on Blackboard.
    • Notes/Handouts: If I have time during the semester, then I may write some notes or handouts to give as reading assignments.
    • Textbook: You don't need a textbook for this class, but much of the material is based on Object-Oriented Programming in Java, A Graphical Approach, by Kathryn E. Sanders and Andries van Dam. The university's library should have a copy of that book if you would like to read it. I may also post small portions of the book on Blackboard for reading assignments.
  • Lecture Notes: Some classes will be more traditional lectures in which I cover examples and write notes on the board. You should take your own notes during class and ask a classmate for their notes if you're absent.
  • Class Activities/Examples: We will have practice activities, coding demonstrations, and reviews during class. Activities will be posted before class, and any material (code, diagrams, etc.) created during class will be posted after that class when possible.
  • Videos: I may create some video lectures and tutorials, but I probably won't have time to do this for most topics.

Software

  • Text Editor/IDE: I recommend Eclipse for SER120 and SER120L, but you're free to use any editor you're comfortable with. Assignment directions will assume you are using Eclipse, and you should be prepared to figure out how to perform the same tasks in your own editor if you choose not to use Eclipse. Don't expect me to help you troubleshoot another editor or IDE.
  • Java: You should have Java installed from CSC110. If not, then please install the JDK for a recent version of Java (as of 2025, Java 23 is ideal).
  • diagrams.net: This is a web app for creating flowcharts and other diagrams. It can also integrate with Google Drive and allow multiple people to edit the same diagram, but some students have reported issues with lag while doing this.

Other Resources

The Java API is a useful reference. It may be intimidating right now, but the more you learn about Java and programming in general, the more easily you'll be able to navigate the API and use it to look up information.

Finally, asking good questions is one of the most useful skills you can have as a student. In my experience, most people (myself included) don't ask questions as often as they should, and it can be difficult to ask a question that will quickly lead to a helpful answer. I think the blog posts below provide excellent advice for asking and answering questions:

  • https://jvns.ca/blog/good-questions/
  • https://jvns.ca/blog/2021/10/21/how-to-get-useful-answers-to-your-questions/
  • https://jvns.ca/blog/answer-questions-well/
