Using Other Sources

By default, all work submitted in this class must be entirely your own work. This section covers the requirements for using work that is not yours in an assignment.

Academic Integrity Requirements

First, the following are always acceptable:

  • Code or other material intentionally provided with an assignment can be used freely, and you don't need to include any credit for this material.
    • If material was obviously included by accident (for example, an entire solution was posted with the assignment), then submitting the material as though it was your own work may be treated as cheating or plagiarism.
    • If material was included by accident (obvious or not), then I will remove the material as soon as I am aware of it and notify the class. If you used the material assuming it was posted intentionally, then you will be allowed to update and resubmit the assignment without that material.
  • If I provide material outside of an assignment, such as my notes or an in-class example, then you are free to use it in any assignment for this class. The material should still be credited as mentioned below.

Otherwise, to comply with the AI policy when using other sources:

  • If copying, paraphrasing, or basing your work on material from another source:
    • You must have my written permission (an email is sufficient) to use that material in your assignment. I may grant permission in the assignment directions, but if not then you'll need to ask me ahead of time.
    • All work that is not your own must be clearly credited. You must specify which part(s) of the assignment are from another source, which source it was from, and link to or otherwise provide a means to check the original source.
  • If you are using another source for help without copying, paraphrasing, or basing your work on the material in that source:
    • You must credit the source as you would credit someone who provided assistance, as noted in the Asking for Help section.

Last, a note about "double-dipping" and self-plagiarism:

To quote the university's AI policy, double-dipping refers to "multiple uses of the same work" or "presenting the same or substantially the same written work (or portion thereof) as part of the course requirement for more than one project or course, without the express prior written permission of the instructor(s) involved." I'm usually okay with you reusing work in other classes or using work from another class in my class, but you still need written permission from both me and the other instructor.

If you are retaking this class, then I expect you to complete every assignment from scratch without reusing any of your old work. I consider reusing work from the last time you took this class to be another form of double-dipping.
