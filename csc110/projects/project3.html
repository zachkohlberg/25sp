Project 3: Dice Game Part 1

This program will simulate dice rolls and allow the user to reroll some of those dice.

Primary Topics

  • Simple conditional expressions
  • Simple iterative expressions
  • Simple functions
  • Simple data structures (arrays, strings)

Prerequisites and Setup

  • Create a file named DiceGame1.java in your projects folder.
  • Create a file named Dice.java in your projects folder and copy the code below into this file. You are required to use the functions in Dice.java to generate all random numbers in your program.
/**
 * Simulates dice rolls with Math.random(). Includes convenience methods for
 * polyhedral dice (d4, d6, d8, d10, d12, d20).
 *
 * This class is included as starter code so that my tests can replace it with a
 * fake random number generator that supplies a predetermined series of numbers
 * instead of simulating dice rolls. This makes it easier to write test cases,
 * but it means you MUST use these functions to generate your random numbers.
 *
 * @author Zach Kohlberg
 */
public class Dice {
    public static int roll(int sides) {
        return (int) (Math.random() * sides) + 1;
    }

    public static int d4() {
        return roll(4);
    }

    public static int d6() {
        return roll(6);
    }

    public static int d8() {
        return roll(8);
    }

    public static int d10() {
        return roll(10);
    }

    public static int d12() {
        return roll(12);
    }

    public static int d20() {
        return roll(20);
    }
}

Tasks

Task 1: Guided Coding

Follow the steps below and write the program as directed. Test your code frequently, at the bare minimum performing every test given in the steps.

If your code is not working as expected, do not move on to the next step until you have fixed it. Moving on without finishing a step correctly will make the problem significantly harder to fix and likely add new problems as you continue working. If you disregard this advice, then you may get to the end of the steps and find it would actually be less work to start over from step 1 than to debug the code you just finished writing.

  1. This program is broken into several functions. We'll write each function separately, test it in main, and then write the real main function once we've finished all of our separate functions. Take a look at the Sample Run section to see what the program looks like when running before you continue.

  2. Our first function, printDice, will display the dice as shown in the sample run. This involves two steps:

    • Print the labels above the dice, and remember to include two spaces between each letter
    • Loop over the dice array and print each element followed by two spaces

    Remember that we can process an array with a for loop like this:

    for (int i = 0; i < array.length; i++) {
    // here we would do something with array[i]
}

    To test printDice, try calling it in your main function. You can hard-code a couple of arrays and check whether the correct numbers appear in your output. For example:

    void main() {
    printDice(new int[] { 1, 2, 4, 2, 3 });
    printDice(new int[] { 6, 5, 5, 6, 5 });
}

    You can remove this test code from main once you're confident that printDice produces the correct output.

  3. Next we'll write the rollAllDice function. This function should set each element of an int array equal to a random number from 1 to 6. You already have two methods you can call from the Dice class to generate these numbers (Dice.roll(6) and Dice.d6()), and the previous step covered iterating over an array.

    To test this function, you'll want to print an array before and after calling rollAllDice. For example:

    void main() {
    int[] array = new int[5];
    printDice(array);
    rollAllDice(array);
    printDice(array);
}

    We should see zeroes when we first print the array (elements of a newly-created int array default to zero), and we should see different numbers from one to six the second time we print the array. Running the program multiple times should produce different results.

  4. The rollSomeDice function is similar to the rollAllDice function, but we need to choose which dice to roll based on the String parameter. See the rollSomeDice section for details.

    You can copy your rollAllDice function to get started, but don't forget to add the second parameter. You also need to add an if statement inside the loop to decide whether to reroll the die at that index.

    There are a few ways you could check whether a die needs to be rerolled. One approach you can try:

    • Convert the index to a letter by adding it to 'a' and converting the result back to a char. For example, (char)('a' + 2) would give you the letter 'c'.
    • Use the String parameter's .contains(...) method to check whether it contains this character. For example, "abc".contains('b') would return true and "abc".contains('e') would return false.

    You'll want to test this function more thoroughly than the previous two. Try calling it with several different strings, and run the program multiple times. Remember that randomizing a number won't always changes it! You might reroll a die whose value was three and end up rolling another three. If a number you rerolled stays the same, that doesn't necessarily mean your code isn't working. This makes it important to run the program multiple times when testing.

  5. Now it's time to finish the program by writing your main function. Your main should perform the following steps:

    • Print a welcome message
    • Create a dice array and roll all of the dice
    • Repeat until the user enters the command "quit":
      • Print the dice
      • Prompt the user for a command
      • Unless the user enters "quit", assume their command is "r" (abbreviating "roll") followed by a space and one or more letters from a to e. For example, "r abcde" or "r ebc".
      • Use your rollSomeDice to reroll the dice listed in the input
    • Print a goodbye message

Random Numbers

To ensure compatibility with my tests, there are a few additional requirements whenever you have to generate a random number:

  • Each random number is generated by either the roll function or the d6 function of the Dice class.
  • Your program "rolls" the dice in the order they appear in the array.
  • Your program does not roll extra dice. For example, calling one of the Dice class's functions to generate a random number when you don't need to roll a die, or rolling each die twice instead of once.

Functions

Your program must include all of the functions described below.

printDice

  • Return type: void
  • Parameters:
    • An int array representing a set of five dice

Print the dice with their labels as shown below. The numbers used below are an example and would only be correct if the array contained the numbers 1, 3, 2, 2, and 6 in that order.

A  B  C  D  E
1  3  2  2  6

rollAllDice

  • Return type: void
  • Parameters:
    • An int array representing a set of five dice

Each element of the array parameter must be set to a random number from 1 to 6.

rollSomeDice

  • Return type: void
  • Parameters:
    • An int array representing a set of five dice
    • A String containing one or more letters from a to e, representing the labels of all dice that will be rerolled

Each element of the array parameter whose label appears in the string parameter must be set to a random number from 1 to 6. Each element of the array parameter whose label does not appear in the string parameter must be left as is.

Labels are not case-sensitive, but your code will only be tested with lowercase letters. An element's label is determined by its index:

  • index 0 -> a
  • index 1 -> b
  • index 2 -> c
  • index 3 -> d
  • index 4 -> e
