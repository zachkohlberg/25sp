Notes for Learning to Program

Text Editor

When we write code, we are editing plain text files. A text file only contains text with no additional formatting (no font sizes, colors, headers, paragraphs, etc.). Programs such as Microsoft Word and Google Docs deal with formatted text and are not suitable for editing code. As a programmer, you will need software designed to edit text files, ideally one meant for programming. Luckily, there are many such editors available for free.

Install Geany

The recommended editor for this class is Geany (https://www.geany.org/). It has all the features you'll need for this class without the bloat of VS Code (a more popular editor that I used to recommend). You can download and run the installer from the Geany website.

Preferences

Plugins

Go to Tools -> Plugin Manager and check the File Browser, Save Actions, and Split Window plugins. These will enable some useful features.

The file browser plugin adds a files tab to the sidebar. This lets you quickly navigate directories and open files without going through the normal File -> Open dialog for each file.

The save actions plugin includes auto save and backup copy features than can help prevent lost work. You'll need to configure this in Plugin Settings. I recommend checking "enable save when losing focus" and "enable". You can change the auto save interval if you want it to save more or less frequently.

The split window plugin allows you to open two files side by side. It's often convenient to see more than one file at a time. You can turn this on by selecting Tools -> Split Window and selecting either option.

Startup Path

In Preferences, you can set the startup path to the workspace you created earlier. Geany will start in this directory rather than your home directory, which you'll probably find more convenient.
