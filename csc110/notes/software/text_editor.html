Text Editor

Using Geany

For now, you only need to know how to perform a few tasks in your editor:

  • Create, save, and open files (File -> New, Save, and Open)
  • Edit the text in a file

I recommend learning more than the bare minimum. The more you know about your editor, the faster and more convenient it becomes to edit code. Take a look at the commands available in the various dropdown menus (particularly File and Edit) and learn what some of them do. These menus also show keyboard shortcuts for most commands. Try getting in the habit of using keyboard shortcuts instead of reaching for the mouse.

File Extensions

A text file normally has the file extension .txt. However, the file extension doesn't change anything about the file's contents. It's just there to tell other software and the user what kind of information is stored in the file. When we write code, we give our file a different extension to indicate what kind of code we are writing. For example, Python files end in .py, Java files end in .java, and C++ files end in .cpp. These are all plain text files, and changing the file extension would not convert the text inside the file to a different format.

When you save a new file for the first time, make sure you use the correct file extension. Since we're using Java, this will usually be .java.

Neovim

I use a text editor called neovim (https://neovim.io/). It has a steep learning curve and I do not recommend it for new programmers, but you will see me using it in videos and during class. Nothing we do in this class is specific a particular editor, and anything I show you in neovim is still a task that you can replicate using Geany. It doesn't matter which editor you're watching someone else use as long as you know your own editor well enough to use it effectively.
