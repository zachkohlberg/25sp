Description

CSC110 is an introduction to programming using the Java programming language. The primary focus of this class is developing problem-solving skills and a general knowledge of programming that will be useful to you in most programming languages. While we will spend a lot of time learning about Java and how to write good code, writing code isn't our ultimate goal. The most important skill you can learn in this class is how to think like a programmer. Once you've mastered that, writing code is surprisingly easy.

Objectives

CSC110 and CSC110L share objectives. Upon successful completion of both courses, students should be able to:

  1. Discuss the challenges inherent in communicating with (programming) a (non-reasoning) computer.

  2. Break a complex problem down into smaller more manageable components.

  3. Provide step-by-step instructions to solve small computational problems.

  4. Using at least one programming language, write examples of and solve problems in programs with the following basic programming constructs and tools:

    1. Mathematical expressions
    2. Conditional expressions
    3. Simple iterative statements
    4. Simple functions
    5. Simple data structures, such as arrays, lists, and strings

  5. Discuss the importance of commenting and good code structure/style
