Exercise 1: Terminal Navigation

Download and unzip sample_dir.zip. Use the cd command to navigate to your downloads folder and enter the newly unzipped Quinnipiac directory.

From the start directory, use a series of cd commands to visit the following directories in order:

  1. TH130
  2. LearningCommons
  3. CCE030
  4. Quinnipiac (return to the starting directory)

Once you're able to visit all of these locations, write down the cd commands you used. Answer the following questions:

  1. What's the smallest number of commands you could have used to visit all of these locations?
  2. What's the largest number of commands you could have used to visit all of these locations? For a command to count, it must take you closer to the next location you're trying to visit.
  3. If you make use of the tab key, how few letters could you type when writing the first cd command to move from Quinnipiac to TH130?

Exercise 2: Exploring Java

Below are three example Java programs. Copy each program to a new file in your workspace and try running it. If you're unable to run a program, then ask for help! All programs should work as long as you're using Java 23 with preview features enabled.

Program 1: Hello.java

void main() {
    println("Hello Zach, welcome to CSC110!");
}

Program 2: FullName.java

void main() {
    var firstName = readln("What is your first name? ");

    var lastName = readln("What is your last name? ");

    println("Your full name is " + firstName + " " + lastName + ".");
}

Program 3: Add.java

void main() {
    var x = Integer.parseInt(readln("Please enter an integer. "));

    var sum = x + 1;
    println(sum);
}

Once you've run each of the programs, try making the following changes:

  1. Make Hello.java greet you instead of your professor.
  2. Make FullName.java print the user's name in "Last, First" format instead of "First Last". For example, "Kohlberg, Zach" instead of "Zach Kohlberg".
  3. Make Add.java read two numbers from the user and print their sum instead of reading a single number and printing that number plus one.

We haven't explained what this code does yet, but I want you to try figuring it out based on the working examples. It's okay if this takes some trial and error. We'll learn more about how these programs work next week.
